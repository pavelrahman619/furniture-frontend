# Product Overview

## Furniture E-commerce Platform

This is a modern furniture e-commerce application built with Next.js, providing a comprehensive online shopping experience for furniture products.

## Key Features

- **Product Catalog**: Browse and search furniture products with advanced filtering
- **Shopping Cart**: Full cart functionality with localStorage persistence and backend synchronization
- **Order Management**: Complete order tracking system with real-time status updates
- **User Authentication**: Login/registration with cart synchronization across sessions
- **Admin Panel**: Administrative interface for product and order management
- **Responsive Design**: Mobile-first approach optimized for all devices

## Business Logic

- **Guest Shopping**: Users can browse and add items to cart without authentication
- **Cart Persistence**: Cart items persist across browser sessions via localStorage
- **Cart Synchronization**: Guest carts merge with user accounts upon login
- **Order Tracking**: Customers can track orders using order numbers
- **Real-time Updates**: Integration with backend APIs for live data

## Target Users

- **Customers**: Browse, purchase, and track furniture orders
- **Administrators**: Manage products, orders, and customer data
- **Guest Users**: Browse products and maintain cart without registration
